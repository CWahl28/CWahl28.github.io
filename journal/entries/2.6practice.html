2.6: Employ code, pre, kbd, and samp tags to display preformatted text, format code snippets and user input.

Summary

Tags used to display text, and communicate things to the user and browser.

They are mostly semantic tags, they mostly mean things to the browser, not the user.

Parts

  • Code
  • Pre
  • kbd
  • Samp

Code: displays inline code snippets. Displays monospaced font. Used to display code in the middle of a paragraph.

Pre: displays preformatted text. Displays monospaced font and preserves all characters around it.

Kbd: displays keyboard input. Displays monospaced font and sometimes a box around the text.

Samp: displays sample output. Displays monospaced font and sometimes emphasized in some way.

kbd and samp are good for accesibilty, can make a page easier to read for users and specifies content on a page
